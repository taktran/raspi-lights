# raspi lights

A minimal set of files to get a Neo pixel light strip working on the raspberry pi.

## Development

On a raspberry pi:

* Install node (tested on node v5.9.1)
* Install deps: `npm install`
* Run

        sudo node index.js
        # or
        sudo node lights-backpack.js
